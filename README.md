# AI Ephemeris Data: 10min 2020-2030

Static daily JSONL/GZIP ephemeris files for AI lookup.

- Cadence: 10 minutes
- Date range: 2020-01-01 to 2030-12-31 UTC
- Files: 4018
- Records per day: 144
- Coordinates: geocentric apparent tropical ecliptic longitude and latitude

Start with index.json, then download one daily file from data/10min/YYYY/YYYY-MM-DD.jsonl.gz.

This branch contains generated data only. The main documentation and generator are in the default branch.
